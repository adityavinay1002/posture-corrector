# PosturePal - AI-Powered Posture Assistant

**PosturePal** is a privacy-first, real-time posture monitoring application that helps you maintain healthy ergonomics while working. Powered by **MediaPipe** computer vision, it runs entirely in your browserâ€”no video data is ever sent to a server.

## Features

- **Real-Time Analysis**: Detects slouching, head tilts, and distance from screen instantly.
- **Privacy-First**: All processing happens locally in your browser.
- **Smart Alerts**:
  - **"Sit Straight"**: Detects uneven shoulders or spinal compression (backbone slouching).
  - **"Move Back"**: Warns you if you lean too close to the screen.
  - **Head Turn Detection**: Understands when you are just looking sideways vs. slouching.
- **Background Monitoring**: Keeps watching and notifies you even when you switch tabs.
- **Customizable Sensitivity**: Fine-tune the detection strictness to your preference.

## Technologies

- **React + TypeScript**: Robust and type-safe UI data flow.
- **Vite**: Ultra-fast build tool.
- **MediaPipe Pose**: State-of-the-art on-device machine learning for body tracking.
- **Tailwind CSS**: Modern, responsive styling.
- **Framer Motion**: Smooth, beautiful animations.
